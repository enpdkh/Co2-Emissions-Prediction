# CO2 Emissions Prediction of Vehicles

## Overview
Machine learning project to predict vehicle CO2 emissions 
using the Canadian NRCan dataset (7,385 vehicles, 2000–2022).

## Models
- Multiple Linear Regression — R² = 0.9910
- Polynomial Regression (deg=4) — R² = 0.9970

- `Mini_Project_AI_Notebook.ipynb` — Full notebook
- `Mini_Project_AI_Report.pdf` — Scientific report
